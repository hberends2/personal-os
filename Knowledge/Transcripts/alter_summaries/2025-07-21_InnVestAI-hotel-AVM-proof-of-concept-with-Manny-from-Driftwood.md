### Product Features

- Deal path and pipeline management:
	- Property details input with required fields
	- Options for grouping, filtering, sorting properties
	- Versioning capability for different JV partner scenarios
- Department and item selection:
	- Customizable categories for F&B, operations, etc.
	- Controls what appears in pro forma to streamline workflow
- Document upload:
	- Central repository for all deal documents
	- Document data extraction using AI for financial statements, STR reports
	- Future goal to eliminate manual uploads through direct data connections
- Market data management:
	- Ability to upload and edit STR reports
	- Custom input for operating statistics and transaction data
	- Plans for automated data collection via public records
- Acquisition modeling:
	- Forward-looking valuation and pro forma sections
	- Predictive analytics for market trends
	- Always-visible key metrics during model building
	- Multiple forecasting methods (direct input or growth percentages)
	- Detailed capital expenditure planning

### Data Strategy

- Anonymized and aggregated data model:
	- 78% participation rate with similar model at Red IQ
	- “Pay to play” access to comparative reports
	- Will not share property-specific information, only aggregated data
- Future data partnerships:
	- Working on strategic relationships with hotel data providers
	- Exploring integration with property management systems
	- Potential partnership with MindClick for PIP (Property Improvement Plan) tracking
	- Title company integration for closing data

### Competitive Positioning

- Positioned as deal-focused platform vs. competitors’ operational focus:
	- “Bespoke version for hotel industry built by hotel investors of deal path plus an AVM”
	- Single Pane (Michelle Russo) and CHM Warnick focus on operational asset management
	- ProfitSoar recently launched ProfitOwner with operational focus
