### Product Demo & Features Overview

- Core investment underwriting/valuation tool targeting investors, brokers, lenders, mortgage servicers, asset managers
- Pipeline feature for tracking multiple deals throughout ownership lifecycle
- Customizable revenue/expense categories - select only relevant items (food & beverage, room service, etc.)
	- Ellen noted banquets/catering missing from dropdown (confirmed it’s included in final version)
- Document upload capability for financial statements, offering memorandums, STR reports
	- Auto-populates historical data using AI extraction
- Market data section with editable STR data, operating statistics, transaction comparables
- Flexible input methods for all assumptions (acquisition, financing, equity partnerships, exit)
	- Partnership waterfall functionality for multiple investors
	- Detailed vs. summary view options for closing costs
- Live KPI cards update instantly with assumption changes
- Pro forma projections with multiple forecasting methods (flat growth, year-by-year, cost per occupied room, % of revenue)

### Key Differentiators & Roadmap

- AI data extraction currently achieves 85-90% model pre-population from uploaded documents
- Planned API integrations with 300+ data sources
	- Real estate tax data (Ryan, CBRE)
	- CapEx integration with Procore for renovation tracking
	- Management company integrations (Drew mapped 50 companies’ chart of accounts)
	- Direct data transmission from operators like HEI
- Portfolio comparison capabilities with consistent data formatting across properties
- Goal: Reduce acquisition underwriting from 2 days to 20 minutes-1 hour
